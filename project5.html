Automated Plant Monitoring System

Interfacing a Micro-controller to External Sensors

  • Category: Computer Interfacing
  • Project URL: Github

    • Arduino Uno R3 YL-69 Soil Moisture Sensor MQ4 Methane Gas Sensor TMP36 Temperature Sensor LDR Sensor

This project introduces an Automated Plant Monitoring System (APMS) utilizing sensors to enhance agricultural practices. Key sensors include the YL-69 Soil Moisture Sensor, which detects soil conditions and triggers watering when needed. The MQ4 Methane Gas Sensor identifies methane levels, crucial for plant health assessment. The TMP36 Temperature Sensor monitors environmental temperature, activating alerts if conditions exceed optimal ranges. Additionally, the LDR Sensor measures sunlight exposure, aiding in determining adequate light levels for plant growth.
